- [x] 상품 결제
- [x] 랜덤쿠폰 발급
- [x] 쿠폰 사용

# ❗️ 예외 상황 ❗

## 공통예외
- [x] 공백을 입력한 경우 ex) `' '`

## 쿠폰 관련 예외
- [x] 쿠폰이 존재하는데 발급받는 경우

## 주문 관련 예외
- [x] 존재하지 않는 상품명을 입력한 경우 ex) `드라이버` 
- [x] 수량을 초과하여 입력한 경우 ex) `후드티,300`
- [x] 주문 포맷이 잘못된 경우 ex) `바지.300` , `반팔:12`, `바지-12`, `300,바지`

## 결제 관련 예외
- [x] 숫자 외의 금액을 입력한 경우 ex) `만원`, `2만원`
- [x] 총 금액보다 적은 금액을 입력한 경우 ex) `100`
- [x] 쿠폰이 존재하지 않는데 쿠폰을 사용하려는 경우
- [x] 메뉴 번호 외의 값을 입력한 경우 `0`, `일번`, `30`
- [x] 포인트가 존재하지 않는데 포인트를 사용하려는 경우]
- [x] 포인트가 최종금액보다 많이 있는경우

## 장바구니 관련 예외
- [x] 존재하지 않는 상품 삭제하려는 경우 ex) `반티,1`
- [x] 장바구니에 담긴 수량보다 많은 수량을 삭제하려는 경우 ex) `반팔티,300`
- [x] 0개의 수량을 삭제하려는 경우 ex) `반팔티,0`
- [x] 삭제 포맷이 비정상적인 경우 ex) `반팔티-300`, `반팔티:300`
