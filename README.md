# 💸 기능 💸
### 메인 메뉴
```
안녕하세요. 무무 쇼핑몰 입니다.

1. 상품 구경하기
2. 랜덤쿠폰 발급받기
3. 장바구니 보러가기
4. 포인트 확인하기
5. 주문내역 확인하기
6. 쇼핑 그만하기
메뉴중 하나를 선택해주세요. (ex. 1)
```

### 상품 구경하기
```
상품을 구경하고 맘에드는걸 장바구니에 넣어보세요.

[Top]
-----------------------------
- [Printing Panel T-Shirt] S사이즈, 60,000원, 5개 남음 
- [Oversize Hooded Sweatshirt] M사이즈, 150,000원, 3개 남음 
- [Curved Angora Knit] L사이즈, 130,000원, 2개 남음 

[Bottom]
-----------------------------
- [Wide Denim Pants] S사이즈, 180,000원, 3개 남음 
- [Straight Denim Pants] M사이즈, 90,000원, 10개 남음 
- [Cut-off Tapered Sweatpants] L사이즈, 150,000원, 6개 남음 

아래의 메뉴를 선택해주세요.
1. 상품 상세보기
2. 장바구니 담기
```

### 상품 상세보기
```
상품 상세를 보고싶은 상품명을 입력해주세요. (ex. 반팔티)
Straight Denim Pants

--------------------------------
[Straight Denim Pants]의 상세정보 입니다.
--------------------------------
사이즈 - M
기장 - 104cm
허리둘레 - 41.0cm
밑위 - 31.0cm
허벅지 - 32.0cm
밑단 - 22.0cm
남은 수량 - 10개
가격 - 90,000원
```

### 장바구니 담기
```
장바구니에 담을 상품명과 수량을 입력해주세요. (ex. 반팔,1)
Wide Denim Pants,1

[Wide Denim Pants] 상품 1개가 장바구니에 추가되었습니다.
```

### 쿠폰 발급받기
```
축하드립니다. 15% 쿠폰을 발급해드렸습니다.
유효기간은 5분 입니다.
```

### 장바구니 살펴보기
```
장바구니속 상품입니다.
- [Wide Denim Pants] S사이즈 1개 180,000원

총 가격 : 180,000원
-------------------------------------------------------
```

### 장바구니 아이템 삭제하기
```
제거하고 싶은 상품의 상품명과 수량을 입력해주세요. (ex. 연청바지,1)
Wide Denim Pants,1

-------------------------------------------------------
[Wide Denim Pants] 1개가 정상적으로 삭제되었습니다.
-------------------------------------------------------
장바구니속 상품입니다.
- [Wide Denim Pants] S사이즈 2개 360,000원

총 가격 : 360,000원
-------------------------------------------------------
아래 메뉴를 선택해주세요.
1. 결제하기
2. 장바구니 상품 제거
3. 홈으로 돌아가기
```

### 결제하기
```
-------------------------------------------------------
결제하실 상품 리스트 입니다.
-------------------------------------------------------
- [Wide Denim Pants] S사이즈 1개 180,000원

쿠폰 할인 금액 : 0원
포인트 적용 : 0원
총 가격 : 180,000원
-------------------------------------------------------
사용가능한 쿠폰 : 쿠폰 없음 / 15% 쿠폰
사용가능한 포인트 : 1,000원
아래 메뉴를 선택해주세요.
1. 쿠폰 사용하기
2. 포인트 사용하기
3. 결제하기
3

총 가격은 151,000입니다.

돈을 지불해주세요.
160000

지불한 금액 : 160,000
잔돈 : 9,000
포인트 : 15100

구매 감사합니다. 계속 쇼핑하시겠어요? (y / n)
```

# ❗️ 예외 상황 ❗

## 공통예외
- [x] 공백을 입력한 경우 ex) `' '`

## 쿠폰 관련 예외
- [x] 쿠폰이 존재하는데 발급받는 경우

## 주문 관련 예외
- [x] 존재하지 않는 상품명을 입력한 경우 ex) `드라이버` 
- [x] 수량을 초과하여 입력한 경우 ex) `후드티,300`
- [x] 주문 포맷이 잘못된 경우 ex) `바지.300` , `반팔:12`, `바지-12`, `300,바지`

## 결제 관련 예외
- [x] 숫자 외의 금액을 입력한 경우 ex) `만원`, `2만원`
- [x] 총 금액보다 적은 금액을 입력한 경우 ex) `100`
- [x] 쿠폰이 존재하지 않는데 쿠폰을 사용하려는 경우
- [x] 메뉴 번호 외의 값을 입력한 경우 `0`, `일번`, `30`
- [x] 포인트가 존재하지 않는데 포인트를 사용하려는 경우]
- [x] 포인트가 최종금액보다 많이 있는경우

## 장바구니 관련 예외
- [x] 존재하지 않는 상품 삭제하려는 경우 ex) `반티,1`
- [x] 장바구니에 담긴 수량보다 많은 수량을 삭제하려는 경우 ex) `반팔티,300`
- [x] 0개의 수량을 삭제하려는 경우 ex) `반팔티,0`
- [x] 삭제 포맷이 비정상적인 경우 ex) `반팔티-300`, `반팔티:300`
